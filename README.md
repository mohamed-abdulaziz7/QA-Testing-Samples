# QA Testing Samples

## 📌 Overview
This repository showcases a QA testing portfolio demonstrating hands-on experience in manual and API testing across different application flows.

Key areas covered:
- Requirement-based test case design  
- Exploratory testing on a real-world e-commerce application  
- Bug reporting with clear reproduction steps and evidence  
- End-to-end test planning for checkout processes  
- API testing and validation for backend services  

This work reflects strong understanding of testing methodologies, defect reporting, and quality assurance practices.

---

## 🧪 Testing Approach
- Designed test cases based on user stories and acceptance criteria  
- Performed manual exploratory testing for web flows  
- Documented defects including:
  - Reproduction steps  
  - Actual vs expected results  
  - Severity & priority  
  - Screenshots
